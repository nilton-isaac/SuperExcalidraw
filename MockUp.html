Getting Started

Welcome to Synth! This is your all-in-one workspace for documentation and visual collaboration.

Overview

Synth combines the best of both worlds: structured documentation and freeform whiteboarding. Create beautiful docs on the left, and bring them to life with diagrams on the right.

💡 Pro Tip
Double-click anywhere on the whiteboard to create a new element. Use the toolbar to switch between different element types.

Core Components

The platform consists of three main components:

  • Documentation Panel - Structured, Notion-style docs
  • Whiteboard Canvas - Infinite canvas for diagrams
  • Code Blocks - Executable HTML/CSS/JS snippets
🔗 Linking Docs to Diagrams

Select any text in your documentation and drag it to the whiteboard to create a linked reference. Changes sync automatically.

Keyboard Shortcuts

  • Space + Drag - Pan canvas
  • Ctrl + Scroll - Zoom in/out
  • Delete - Remove selected element
  • Ctrl + D - Duplicate selection
  • Ctrl + / - Toggle code block
