# Synth

Whiteboard + documentation workspace built with React, TypeScript and Vite.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Vercel

This app is ready to deploy as a Vite project.

Recommended setup in Vercel:

1. Import the repository.
2. Set the Root Directory to `figmadoc`.
3. Keep the default install command.
4. Use `npm run build` as the build command.
5. Use `dist` as the output directory.

There is also a local [vercel.json](./vercel.json) with SPA rewrite support for direct routes.
